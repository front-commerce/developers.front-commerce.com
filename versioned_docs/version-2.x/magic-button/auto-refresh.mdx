## How to activate Auto-refresh

To activate the **Auto-refresh** feature, click on the Magic Button and select
the Auto-refresh mode.

<div style={{ width: 600, maxWidth: "100%" }}>
  <BrowserWindow noPadding>
    <img
      src={require("./assets/enable-auto-refresh.gif").default}
      width="100%"
      height="100%"
      style={{ objectFit: "contain", marginBottom: -10 }}
    />
  </BrowserWindow>
</div>

When enabled, the page is continuously updated and then contributors see their
changes as soon as possible.
