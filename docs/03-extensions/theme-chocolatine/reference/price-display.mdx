## Available options

The `theme:priceDisplay` public configuration has the following options:

- `variants`: the list of price variants to display, available variants are
  `priceInclTax` and `priceExclTax`.
- `appearance`: the appearance of the price, available appearances are `labeled`
  and `default`.
  - `labeled`: this includes the label `incl. tax` or `excl. tax`
  - `default`: this does not include any label
